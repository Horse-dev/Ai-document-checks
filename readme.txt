AI-powered document classification system with JetFormBuilder integration for WordPress.

== Description ==

Document Classification Plugin is a powerful tool that uses AI to automatically classify uploaded documents based on learned patterns. It integrates seamlessly with JetFormBuilder to enhance your form workflows.

= Features =

* **Upload Component:** A custom Gutenberg block that adds a document upload field with progress indicator and validation
* **Admin Dashboard:** Interface for admins to upload training documents, add keywords, and document types
* **AI Classification:** System that scans uploaded documents for predefined keywords to identify document types
* **Database Storage:** Structure to store training samples, keywords, document types, and extracted values
* **JetFormBuilder Integration:** Seamless connection with existing form workflows

= Use Cases =

* Automatically classify uploaded documents in forms
* Extract relevant information from documents
* Streamline document processing workflows
* Improve form user experience with intelligent document handling

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/document-classification-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Document Classification screen to configure the plugin
4. Add training documents and document types through the admin dashboard

== Frequently Asked Questions ==

= What document types can be classified? =

The plugin can classify any document type that you train it to recognize. Common examples include birth certificates, driver's licenses, passports, tax forms, and medical records.

= How accurate is the classification? =

The accuracy depends on the quality and quantity of training documents. With a good set of training documents and well-defined keywords, the system can achieve high accuracy rates.

= Does it work with JetFormBuilder? =

Yes, the plugin is designed to integrate seamlessly with JetFormBuilder, adding a custom document upload field that includes classification capabilities.
